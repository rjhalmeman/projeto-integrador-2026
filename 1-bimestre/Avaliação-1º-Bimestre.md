## 🎯 Objetivo

Desenvolver e apresentar um sistema web (parcial).

O projeto terá valor de **5,0 pontos**.

Um dos integrantes da equipe deverá realizar a apresentação para a turma (a escolha é da equipe).

---

## 📦 Funcionalidades Obrigatórias

O sistema deve conter, no mínimo:

- 🔐 **Login de usuário**
  - Validação de credenciais no banco de dados
  - Manutenção da sessão utilizando **cookies**

- 🏠 **Menu principal**
  - Exibir o nome do usuário logado
  - Opção de logout

- 🗂️ **CRUD completo**
  - Implementar **Create, Read, Update e Delete** de **uma tabela do banco de dados**

---

## 🏗️ Estrutura do Projeto

O projeto deve ser dividido em **frontend** e **backend**, seguindo uma organização baseada em **MVC (Model-View-Controller)** ou arquitetura em camadas equivalente.

### 🔧 Backend (Node.js)

Estrutura mínima sugerida:

```
backend/
  server/
  routes/
  controllers/
  models/
  config/
```

- Conexão com banco de dados
- Rotas organizadas
- Controllers separados da lógica de acesso ao banco

---

### 💻 Frontend

O frontend deve ser desenvolvido utilizando **uma abordagens estruturante:**

#### ✔️ Opção 1 (Recomendada)
- React + Vite
- Organização em componentes

#### ✔️ Opção 2 (Simples)
- HTML + CSS + JavaScript puro
- Estrutura por funcionalidade (ex: pasta para cada CRUD)

---

## 🗃️ Banco de Dados

- Utilizar banco relacional (ex: PostgreSQL, MySql, etc) (recomendado)
- O sistema deve realizar consultas reais no banco
